# FrontSkeleton

## Les installations nécessaire
- Node JS : https://nodejs.org/en/download
- Angular : `npm install -g @angular/cli`

## Avant de lancer le projet

Lancer `npm i`

## Pour lancer le projet

Lancer `npm start` et se rendre sur `http://localhost:4200/`

## Fonctionalités

- lire les règles
- hall of fame
- choisir parmi une liste de quiz (monde, continent, langue)
- passer en mode administrateur
- (admin) modifier la liste des quiz jouables
- (admin) ajouter un quiz
- consulter la liste des parties avec un user
